# 离线全屏水印工具

一个纯前端的离线图片水印工具，所有处理在浏览器本地完成，**图片不会上传到任何服务器**。

参考 [wxy97.com/watermark](https://www.wxy97.com/watermark) 的功能重做，并增加了一些实用功能。

## 功能

- 图片上传：点击选择 / 拖拽 / 剪贴板粘贴（Ctrl+V）
- 水印文字：8 种中文预设一键切换，也可自定义
- 水印颜色：颜色选择器 + HEX 输入框 + 9 种预设色板，三方联动
- 不透明度、文字大小、水印间隔、旋转角度，全部可调
- 字体、字重可选
- 模板变量：`{date}` `{time}` `{datetime}` `{random}` `{filename}` 可插入水印
- 满屏平铺 + 交错排列，旋转后四角不留白
- 一键保存为 PNG
- 纯本地处理，关闭页面即清除，零隐私泄露

## 使用方法

1. 下载 `watermark.html`
2. 用浏览器打开（推荐 Chrome / Edge）
3. 拖一张图片到中间区域
4. 左侧调整水印参数
5. 点击「保存图片」导出

## 默认参数

| 参数 | 默认值 |
|------|--------|
| 水印文字 | 仅供内部使用 |
| 颜色 | #FF0000（红色） |
| 不透明度 | 40% |
| 文字大小比 | 4.0 |
| 水印间隔比 | 2.0 |
| 旋转角度 | -30° |
| 字体 | 无衬线 |
| 字重 | 粗体 |

## 技术栈

纯 HTML + CSS + JavaScript，零依赖，单文件，开箱即用。
