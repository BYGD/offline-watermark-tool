# 离线全屏水印工具

一个纯前端的离线图片水印工具，所有处理在浏览器本地完成，**图片不会上传到任何服务器**。适合给敏感截图、内部文档、实名认证资料加水印溯源。

## 功能特性

### 水印内容
- 8 种中文预设一键切换：仅供内部使用 / 内部资料·禁止外传 / 机密文件 / 请勿外传 / 严禁复制 / 仅供XX实名认证使用 / 带日期水印 / 版权所有
- 支持自定义任意水印文字
- 模板变量：`{date}` `{time}` `{datetime}` `{random}` `{filename}` 可自由组合，便于溯源

### 水印样式
- 颜色：颜色选择器 + HEX 输入框 + 9 种预设色板，三方联动
- 不透明度、文字大小比、水印间隔比、旋转角度，全部可调
- 字体（无衬线 / 衬线 / 等宽 / 楷体 / 黑体）与字重可选
- 满屏平铺 + 交错排列，旋转后四角不留白

### 图片处理
- 上传方式：点击选择 / 拖拽 / 剪贴板粘贴（Ctrl+V）
- 一键保存为 PNG
- 纯本地处理，关闭页面即清除，零隐私泄露

## 使用方法

1. 下载 `watermark.html`
2. 用浏览器打开（推荐 Chrome / Edge）
3. 拖一张图片到中间虚线区域
4. 左侧调整水印参数，右侧实时预览
5. 点击「保存图片」导出

## 默认参数

| 参数 | 默认值 |
|------|--------|
| 水印文字 | 仅供内部使用 |
| 颜色 | #FF0000（红色） |
| 不透明度 | 40% |
| 文字大小比 | 4.0 |
| 水印间隔比 | 2.0 |
| 旋转角度 | -30° |
| 字体 | 无衬线 |
| 字重 | 粗体 |

## 技术栈

纯 HTML + CSS + JavaScript，零依赖，单文件，开箱即用。无需安装、无需联网、无需后端。
